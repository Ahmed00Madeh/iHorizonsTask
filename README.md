# iHorizonsTask

## Author Ahmed Madeh

## How to Run the app
    1. Clone this repo
    2. Open terminat window and navigate to project folder
    3. Run pod install
    4. Open iHorizonsTask.xcworkspace and run the project on selected device or simulator

## Build and Runtime Requirements
    . Xcode 12 or later
    . iOS 13.0 or later
    . OS X v11.2.1 or later

## Dependencies
    . ESNetworkManager
    . Kingfisher
    . PromiseKit
    . ObjectMapper
